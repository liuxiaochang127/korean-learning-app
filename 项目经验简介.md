## 项目经验

### 1、延世韩语学习平台

基于 React、TypeScript、Vite、Supabase（PostgreSQL）、TailwindCSS 开发的移动端韩语学习 Web 应用（PWA），核心为本系统是一款面向中国韩语学习者的综合性语言学习平台，围绕延世大学韩国语教材体系构建。核心功能包括韩中双语词典（增删改查+收藏）、基于 SM-2 算法的 SRS 间隔重复记忆系统（智能复习调度）、闪卡复习、单词遮挡背诵、语法知识学习、课文阅读与课程音频播放，同时集成双策略 TTS 语音引擎（本地语音合成 + 有道在线 API 降级）、Supabase Auth 用户认证与 RLS 行级安全数据隔离、资源文件上传管理等辅助功能，并支持 PWA 添加到主屏幕，针对微信内置浏览器及 Android WebView 进行了专项兼容优化。

**行业场景：** 在线教育 / 语言学习。随着中韩文化交流日益密切，韩语学习需求持续增长，传统线下培训存在时间空间受限、学习效率难以量化等痛点。本平台定位于移动端轻量化自学工具，聚焦"词汇记忆 + 语法理解 + 听说训练"三大核心学习环节，通过 SRS 间隔重复算法实现科学记忆曲线管理，利用 TTS 语音合成解决发音纠正需求，以 PWA 技术降低用户使用门槛（无需安装、微信内直接打开），适用于韩语自学者日常碎片化学习场景，同时也可作为培训机构线上辅助教学工具使用。

**功能介绍：**
- **韩中双语词典**：支持关键词模糊搜索、按词性/难度分类浏览，词条包含韩语、罗马音、词性、中文释义、例句及翻译，支持用户自定义增删改查与星标收藏
- **SRS 间隔重复系统**：基于 SM-2 算法根据用户评分（忘记/模糊/记住/熟练）动态调整复习间隔与难度因子，每日自动推送待复习词汇集，逻辑日期以凌晨4点为分界
- **闪卡复习**：正面韩语、背面释义的经典翻转闪卡模式，支持语法卡片与词汇卡片，逐张练习并标记掌握状态
- **单词背诵**：列表化遮挡背诵模式，可选隐藏韩语或中文释义，点击逐词揭示，记录每个单词的复习次数，支持分页加载
- **语法学习**：覆盖初级到高级50+语法知识点，每条包含语法标题、中文说明、韩语例句与翻译，首页每日随机推荐
- **课程音频**：配套教材的课文音频播放器，支持进度条拖拽、播放/暂停控制
- **课文阅读**：韩语课文阅读展示与学习辅助
- **TTS 韩语语音**：双策略语音引擎——优先使用浏览器本地语音合成（SpeechSynthesis API），无韩语语音包或微信环境下自动降级为有道在线TTS API，兼容 iOS/Android/微信
- **资源管理**：支持音频/文本文件上传至 Supabase Storage，文件列表在线预览与播放，中文文件名 Base64 编码处理
- **用户系统**：Supabase Auth 用户认证，个人资料页展示等级、连续打卡天数、学习时长、金币等数据，15天未活跃自动登出
- **PWA 支持**：Service Worker 离线缓存、添加到主屏幕、独立窗口模式，支持 iOS 与 Android 双端安装
